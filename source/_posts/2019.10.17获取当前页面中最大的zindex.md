获取页面中的最大z-index，用于消息通知或者弹出框的显示

<!--more-->

> 获取页面中的最大z-index，用于消息通知或者弹出框的显示

```js
/**
   * @createtime 2019/09/12 15:55:54
   * @author 柯军 <arthaskj@163.com>
   * @desc 获取页面最大z-index
   */
  _getMaxIndex() {
    var maxZ = Math.max.apply(null,
      $.map($('body *'), (e, n) => {
        if ($(e).css('position') !== 'static')
          return parseInt($(e).css('z-index')) || -1;
      }));

    this.maxIndex = maxZ || 10002;

    return maxZ;
  }
```
